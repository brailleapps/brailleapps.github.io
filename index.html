--- title: For Users ---

Applications

Dotify Studio

Dotify Studio is a desktop application for creating, managing and embossing braille. It can create braille from a variety of formats and emboss the results on a wide range of embossers. The application has both professional and end user features making it a good choice for anyone who produces or uses braille.*

To download and use, see the list of releases.

Dotify CLI

Dotify CLI is a command line application for formatting and translating a document into braille* as well as embossing and manipulating PEF-files.

To download and use, see the list of releases.

* Braille translation support in the application is limited out of the box, but can be extended. If you're working for a braille producing organization, ask your IT-department to assist you. If you are an end user, ask your braille supplier if they can help you. Please consider contributing the result back to this project.
