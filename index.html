Provides applications and APIs for producing braille

Applications

Braille Utils

Braille Utils provides a command line interface for embossing and manipulating PEF-files.

To download and use, see the list of releases.

Dotify

Dotify provides a command line interface for formatting and translating a document into braille. Since Dotify extends Braille Utils, it also has access to all the functionality found in Braille Utils.

For developers

API bundles

The following APIs can be used to extend the applications above or to write new ones.

  • braille-utils.api
  • dotify.api
  • dotify.task-api

Implementation providers

The following bundles provide implementations to some part of the APIs above. Typically, they do not have classes intended for direct use. Instead, they can be used by accessing classes found in the APIs.

  • braille-utils.impl
  • dotify.formatter.impl
  • dotify.task.impl
  • dotify.translator.impl
  • dotify.text.impl
  • dotify.hyphenator.impl

Utility bundles

The following bundles provide classes intended for direct use. Many makes use of the APIs above and expects to have access to relevant implementations.

  • braille-utils.pef-tools
  • braille-utils.basic-cli
  • dotify.common
  • dotify.task-runner

Developer tools

  • dotify.devtools
  • dotify.devtools.gui

See also
