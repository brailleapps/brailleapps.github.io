For developers

The wiki project on Github contains lots of useful information, but since you're here, here are some quick facts about the project from OpenHub:

API bundles

The following APIs can be used to extend the applications above or to write new ones.

  • dotify.api

Implementation providers

The following bundles provide implementations to some part of the APIs above. Typically, they do not have classes intended for direct use. Instead, they can be used by accessing classes found in the APIs.

  • braille-utils.impl
  • dotify.formatter.impl
  • dotify.task.impl
  • dotify.translator.impl
  • dotify.text.impl
  • dotify.hyphenator.impl

Utility bundles

The following bundles provide classes intended for direct use. Many makes use of the APIs above and expects to have access to relevant implementations.

  • braille-utils.pef-tools
  • dotify.common

Developer tools

  • dotify.devtools
  • dotify.devtools.gui

See also

For the specifications that these applications are based on, see http://braillespecs.github.io
